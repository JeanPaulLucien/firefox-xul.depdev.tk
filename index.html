The Firefox XUL project was created to save a legacy of Mozilla's past.

The goals:

- provide a source and access to XUL addons
- provide a place to publish new and updated XUL addons
- provide a XUL documentation

The project looks for contributors, check the skill list below to find your:

- javascript (development front-end)
- php (development back-end)
- translators to English, Russian and other languages
- English language inspectors (to check translation)
- in any case, feel free to contact if you want to help or suggest if you want to help contact with us by joinATfirefox-xulDOTdepdevDOTtk

The Firefox XUL project

Почему XUL дополнения?

XUL дополнения, как и браузеры с его поддержкой предоставляют пользователю большую свободу в создании и расширении возможностей браузера. XUL имеет столько широкие возможности, что многие дополнения технически невозможно реализовать в браузерах без поддержки XUL.
